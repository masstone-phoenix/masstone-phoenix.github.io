iOS实现多语言

Foundation提供了以下4个宏来进行多语言的实现，大多数情况下，使用NSLocalizedString替换使用字符串的地方,在Localizable.strings中，写入不同语言对应的字符串即可。

NSString * NSLocalizedString(
  NSString *key,
  NSString *comment
)

NSString * NSLocalizedStringFromTable(
  NSString *key,
  NSString *tableName,
  NSString *comment
)

NSString * NSLocalizedStringFromTableInBundle(
  NSString *key,
  NSString *tableName,
  NSBundle *bundle,
  NSString *comment
)

NSString * NSLocalizedStringWithDefaultValue(
  NSString *key,
  NSString *tableName,
  NSBundle *bundle,
  NSString *value,
  NSString *comment
)

以上4个宏都是对NSBundle的localizedStringForKey:value:table:函数的封装。

但是要实现程序内的多语言切换，需要使用NSLocalizedStringFromTableInBundle这个宏
