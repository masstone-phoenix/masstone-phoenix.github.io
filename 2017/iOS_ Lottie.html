安装

我们的项目没有使用CocoaPods，这里记录一下手动安装的步骤： 主要参考https://github.com/airbnb/lottie-react-native/issues/31

准备： 安装carthage

brew install carthage

第一步，创建Cartfile文件，加入要使用的库，更新

vim Cartfile

carthage update

第二步，在工程中加入库：在Linked Frameworks and Libraries中加入位于 Carthage/Build中的framework。

第三小，写程序。
