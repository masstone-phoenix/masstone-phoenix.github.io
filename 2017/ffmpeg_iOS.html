FFmpeg大家都知道，是用来处理音视频数据的程序。项目中可能要用到，在这记录下编译的过程。参考https://cnbin.github.io/blog/2015/05/19/iospei-zhi-ffmpegkuang-jia/

准备

1.安装yasm。

 cd yasm-1.2.0
 ./configure
 make
 sudo make install
  1. 到https://github.com/libav/gas-preprocessor下载 gas-preprocessor.pl 文件，复制gas-preprocessor.pl到/usr/bin下，修改文件权限 
sudo cp ..../gas-preprocessor.pl   /usr/local/bin/gas-preprocessor.pl
sudo chmod 777 /usr/local/bin/gas-preprocessor.pl

编译

1.下载脚本文件 https://github.com/kewlbear/FFmpeg-iOS-build-script

这个脚本文件自动下载编译。
